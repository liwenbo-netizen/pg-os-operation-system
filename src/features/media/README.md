- media manager workbench;
- Publisher 360;

- commercial test workspace;
- media director readiness command center;

- ecosystem track map for China and APAC media opportunities;
- scored opportunity pool with owners, stages, and next actions;
- outreach and contact proof;
- trusted supply candidate conversion gates;
- onboarding project creation back into the existing Publisher 360 readiness flow.

The current adapter uses in-memory workflow state. Supabase repositories can replace it after migrations are executed.
